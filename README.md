# TaskBot

Telegram-бот делегирования задач. Деплой на сервере через git clone.

## Команды руководителя

- `/new` — поставить задачу одному или нескольким сотрудникам
- `/tasks` — показать открытые задачи с ID
- `/task 12` — показать задачу по ID
- `/edit 12 новый текст` — заменить текст задачи
- `/append 12 уточнение` — добавить уточнение в описание задачи
- `/deadline 12 до пятницы 18:00` — изменить дедлайн
- `/deadline 12 -` — убрать дедлайн
- `/status` — недельная сводка
- `/report` — собрать недельный отчёт и выгрузить его в Google Sheets
- `/sync` — принудительно синхронизировать задачи с Google Sheets
- `/table` — ссылка на таблицу-реестр

Каждая задача получает ID вида `#12`; он показывается в карточках, списках, отчётах и используется во всех командах редактирования.
